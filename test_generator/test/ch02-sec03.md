## 発表用ポスターにおいて「視線誘導」の説明として正しいものはどれか。

## CSSの役割として正しいものはどれか。
- Webサーバーとブラウザ間の通信を制御する
- Webページの文書構造を定義する
- *Webページの文字や画像の大きさ・色・配置などの視覚的なスタイルを定義する
- Webページのフォームデータを処理する

## CSSの書式として正しいものはどれか。
- `タグ名 = プロパティ: 値`
- `プロパティ { セレクタ: 値; }`
- *`セレクタ { プロパティ: 値; }`
- `値 { セレクタ: プロパティ; }`
score: 5
解説: CSSの書式は「セレクタ { プロパティ: 値; }」です。セレクタはデザインを適用する対象，プロパティは変更したい項目，値はその項目をどう変更するかを示します。例：`h1 { color: red; }` はh1見出しの文字色を赤にします。

## 文書において，二列以上に分けて文字や図などをレイアウトすることを何というか。
type: text
answer: 段組み, だんぐみ
score: 10
解説: 段組みとは，文書において二列以上に分けて文字や図などをレイアウトすることです。発表用ポスターで一行の文字数が多いと次の行を見失いやすくなるため，段組みにすると読みやすくなります。段と段の間の余白を十分に取ると情報の構成がわかりやすくなります。

## HTMLで「<」と「>」で挟まれた記号を使って文字列を囲み，文章構造を定義することを何というか。
type: text
answer: マークアップ, markup, mark up
score: 10
解説: マークアップとは，「<」と「>」で挟まれたタグで文字列を囲むことにより，Webページの文章構造を定義することです。例えば「<title>学校紹介</title>」とタグで囲むと，Webサイトのタイトルが「学校紹介」と表示されます。
